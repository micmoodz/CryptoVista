# CryptoVista

CryptoVista is a modern DeFi dashboard web application for managing your crypto portfolio, tracking transactions, and connecting your wallet securely.

## Features
- User authentication (login/signup)
- WalletConnect integration (MetaMask)
- Portfolio overview
- Transaction history
- Protocols tracking
- Settings and dark mode

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/micmoodz/CryptoVista.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend:
   ```bash
   npm start
   ```
4. Start the backend:
   ```bash
   node server.js
   ```

## Technologies Used
- React
- Express (Node.js)
- Tailwind CSS
- Ethers.js
